## Codebase Patterns
- Python 3.12+, Pydantic v2 for all schema definitions
- mypy --strict for type checking
- Package name: docos (Document OS)
- BBox is a type alias for tuple[float, float, float, float] (x0, y0, x1, y1)
- BlockType enum covers all v1 block types including unknown; never drop unknown blocks
- RelationType enum covers all v1 relation types
- DocIR validates unique block_ids via Pydantic field_validator
- All models use Field() with descriptions for self-documenting schema
