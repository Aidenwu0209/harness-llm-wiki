## Codebase Patterns
- This repository is Python-first; use `python -m mypy docos/` for type checking and `python -m pytest tests/ -q` for automated verification.
- Pipeline work is split across `docos/cli`, `docos/pipeline`, `docos/knowledge`, `docos/wiki`, `docos/lint`, `docos/harness`, and `docos/review`; prefer keeping each story focused on one layer.
